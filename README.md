# 🎨 OneDay Pick

원데이 클래스 예약 관리 사이트  
React + TypeScript + Vite 기반으로 제작된 데모 프로젝트입니다.

---

## 📦 기술 스택

| 영역              | 사용 기술              |
| ----------------- | ---------------------- |
| Frontend          | React 18.2, TypeScript |
| 번들러 / 개발환경 | Vite                   |
| 스타일링          | CSS (Custom)           |
| 배포              | GitHub (예정)          |

---

## 🚀 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```
